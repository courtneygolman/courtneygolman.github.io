There are many ways to control the look and readability of a web page with CSS Typography

Examples:

Element CSS Explanation
Line height line-height Spaces the lines of a paragraph in relation to the font size being equal to 1
Word spacing word-spacing Spaces individual words by pixels
Letter spacing letter-spacing Spaces individual letters by pixels
Text transform text-transform Can set all letters to either uppercase or lowercase
Text style text-style To set the font to italic
Font families font-family Can be downloaded, default, or linked to an api; Can set 1-4 values for fallbacks
